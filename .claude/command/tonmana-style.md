# tonmana-style

HTMLワイヤーフレームにトンマナに基づいたスタイルを適用する。

## 使い方

```
/tonmana-style
```

または会話中に「トンマナを適用して」「スタイルを当てて」と伝える。

## 実行手順

以下を順番にユーザーに確認・実行する：

### Step 1: ワイヤーフレームの確認
対象のHTMLファイルパスを確認する。

### Step 2: トンマナのヒアリング
以下の観点でトンマナをヒアリングし、英語プロンプトに変換する：
- ブランドの印象（例：信頼感、革新的、温かみ）
- カラーイメージ（例：ネイビー×ホワイト、アースカラー）
- フォントの雰囲気（例：モダン・サンセリフ、エレガント・セリフ）
- UIスタイル（例：フラット、ミニマル、リッチ）
- ターゲットユーザー

### Step 3: スクリプト実行
```bash
node .claude/mcp/tonmana-style.js <wireframe.html> "<トンマナの英語説明>"
```

### Step 4: 確認・修正
出力された `styled_<ファイル名>.html` を確認し、必要に応じて追加調整する。

## 前提条件

`.env` ファイルに以下が設定されていること：
```
OPENAI_API_KEY=sk-...
ANTHROPIC_API_KEY=sk-ant-...
```
