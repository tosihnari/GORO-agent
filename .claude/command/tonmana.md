# tonmana

デザインHTMLにAI生成のムードボードを使ってトンマナを適用する。

## 実行手順

以下を順番に進める。

### 1. 対象HTMLの確認

カレントディレクトリのHTMLファイルを一覧表示し、どのファイルに適用するか聞く。

### 2. トンマナのヒアリング

以下を聞く：
- どんな印象にしたいか（例：信頼感、エンタメ感、スタイリッシュ）
- カラーイメージ（例：モノトーン×ピンク、ネイビー×ホワイト）
- フォントの雰囲気（例：ゴシック、セリフ、丸ゴシック）
- その他気になることがあれば

ヒアリング内容を英語プロンプトに変換する。

### 3. ムードボード生成

以下のコマンドを実行する：

```bash
node tonmana-generate.js "<英語に変換したトンマナ>"
```

完了したら生成された画像パスをユーザーに伝え、Finderで開くよう案内する：

```bash
open 参考画像/<生成されたファイル名>
```

### 4. 画像確認

「画像を確認しました。このイメージで進めますか？」と聞く。

- **「はい」** → STEP 5 へ
- **「違う」** → トンマナを修正して STEP 3 をやり直す

### 5. スタイル適用

以下のコマンドを実行する：

```bash
node tonmana-apply.js <対象HTML> <参考画像のパス>
```

### 6. 完了案内

出力された `styled_<ファイル名>.html` をブラウザで開くよう案内する：

```bash
open styled_<ファイル名>.html
```

修正希望があれば対応する。

## 前提条件

- `sanari-hp` フォルダに `.env` があること
- `.env` に `OPENAI_API_KEY` と `ANTHROPIC_API_KEY` が設定されていること
